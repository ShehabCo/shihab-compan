# مرة أم سليم - دليل الإعداد والتشغيل

## نظرة عامة

تطبيق ويب فاخر لمطعم عربي يقدم خدمات الطلب والتوصيل مع نظام مصادقة متقدم وتتبع الطلبات في الوقت الفعلي.

## المتطلبات

- Node.js 18+ 
- npm أو yarn
- متصفح حديث بدعم ES6+

## الميزات الرئيسية

### ✅ المصادقة والتحقق
- تسجيل دخول عبر البريد الإلكتروني + كلمة المرور
- تسجيل دخول عبر رقم الهاتف + OTP (SMS)
- تسجيل دخول عبر Google
- التحقق ثنائي الخطوات (2FA)
- إدارة الجلسات الآمنة

### 🍽️ القائمة والطلبات
- عرض 12 طبق عربي فاخر
- سلة شراء ديناميكية
- نظام الولاء (4 طلبات = توصيل مجاني)
- تتبع الطلب في الوقت الفعلي
- إدارة حالة الطلب (الإعداد → التوصيل → التسليم)

### 👨‍💼 لوحة إدارية متقدمة
- إدارة القائمة والأطباق
- مراقبة الطلبات والإيرادات
- إدارة السائقين والمناطق
- تحسين المسارات والتسليمات
- تقارير ومؤشرات الأداء

### 📱 تجربة المستخدم
- واجهة عربية بنسبة 100% (RTL)
- تصميم فاخر وأنيق
- استجابة كاملة للهواتف والأجهزة اللوحية
- أداء سريع وسلس

## التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone <repo-url>
cd mara-umm-salim
```

### 2. تثبيت المتطلبات
```bash
npm install
```

### 3. إعداد متغيرات البيئة
أنشئ ملف `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3000
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

### 4. تشغيل خادم التطوير
```bash
npm run dev
```

ستتمكن من الوصول إلى التطبيق على: `http://localhost:3000`

## المسارات الرئيسية

### للعملاء
- `/` - الصفحة الرئيسية
- `/auth/login` - تسجيل الدخول
- `/auth/signup` - إنشاء حساب جديد
- `/menu` - قائمة الأطباق
- `/cart` - سلة الشراء
- `/orders` - قائمة الطلبات
- `/orders/[id]` - تفاصيل الطلب والتتبع
- `/tracking/[orderId]` - تتبع الطلب الحي

### للسائقين
- `/driver/dashboard` - لوحة السائق
- `/driver/batch` - الطلبات المخصصة

### للإدارة
- `/admin/dashboard` - لوحة المعلومات الرئيسية
- `/admin/dishes` - إدارة الأطباق
- `/admin/orders` - إدارة الطلبات
- `/admin/drivers` - إدارة السائقين
- `/admin/routes` - تحسين المسارات
- `/admin/users` - إدارة المستخدمين

## التطوير والتصحيح

### تفعيل وضع التصحيح
افتح وحدة التحكم في المتصفح (F12) لعرض رسائل السجل التفصيلية.

### عرض OTP المُنشأ
- OTP يُطبع في وحدة التحكم
- OTP يُعرض في رسالة النجاح (في وضع التطوير فقط)
