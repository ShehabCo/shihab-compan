## 🔧 خطوات النشر

### الخطوة 1: تشغيل السكريبتات
```bash
# في Supabase SQL Editor، قم بتشغيل الملفات بالترتيب:
1. scripts/001_create_users_and_profiles.sql
2. scripts/002_create_services.sql
3. scripts/003_create_orders.sql
4. scripts/004_create_wallets.sql
5. scripts/005_create_reviews.sql
6. scripts/006_create_messages.sql
7. scripts/007_seed_data.sql
8. scripts/008_create_functions.sql
9. scripts/009_create_order_functions.sql
```

### الخطوة 2: التحقق من المتغيرات البيئية
تأكد من وجود جميع المتغيرات:
```
SUPABASE_URL
NEXT_PUBLIC_SUPABASE_URL
SUPABASE_ANON_KEY
NEXT_PUBLIC_SUPABASE_ANON_KEY
NEXT_PUBLIC_DEV_SUPABASE_REDIRECT_URL
```

### الخطوة 3: النشر على Vercel
1. انقر على "Publish" في v0
2. أو استخدم GitHub + Vercel:
   ```bash
   git push origin main
   # سيتم النشر تلقائياً
   ```
