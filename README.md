# منصة شهاب - Shahab Marketplace Platform

منصة عربية متكاملة للخدمات الرقمية والمحتوى الإبداعي

## نظرة عامة

شهاب هي منصة سوق رقمية متكاملة باللغة العربية (RTL) تربط بين مقدمي الخدمات الرقمية والمشترين. تتضمن المنصة نظام محفظة متقدم، رسائل مباشرة، إدارة طلبات، وتقييمات شاملة.

## المميزات الرئيسية

### 1. نظام المصادقة والأدوار
- تسجيل دخول وإنشاء حساب بالبريد الإلكتروني
- ثلاثة أدوار: مشتري، بائع، مدير
- التحقق من البريد الإلكتروني
- حماية البيانات بـ Row Level Security (RLS)

### 2. السوق الرقمي
- عرض الخدمات مع البحث والفلترة
- تصنيفات متعددة (تصميم، برمجة، كتابة، تسويق، فيديو، موسيقى، أعمال)
- صفحة تفاصيل الخدمة مع معلومات البائع
- نظام تقييمات ومراجعات

### 3. لوحة تحكم البائع
- إدارة الخدمات (إضافة، تعديل، حذف)
- متابعة الطلبات والمبيعات
- نظام المحفظة مع سجل المعاملات
- عرض التقييمات والإحصائيات

### 4. نظام الطلبات
- إنشاء طلب مع تحديد المتطلبات
- تتبع حالة الطلب (قيد الانتظار، قيد التنفيذ، تم التسليم، مكتمل)
- تسليم الملفات من البائع
- قبول أو رفض التسليم من المشتري

### 5. نظام المحفظة
- رصيد متاح ورصيد معلق
- سجل المعاملات الكامل
- طلبات السحب
- عمولة المنصة 10% تلقائياً

### 6. نظام المراسلة
- محادثات مباشرة بين المشتري والبائع
- ربط المحادثات بالطلبات
- واجهة عربية RTL كاملة
- إشعارات الرسائل الجديدة

### 7. نظام التقييمات
- تقييم من 1 إلى 5 نجوم
- تعليقات اختيارية
- عرض التقييمات على صفحة الخدمة
- إحصائيات التقييمات للبائعين

### 8. لوحة الإدارة
- إحصائيات المنصة الشاملة
- إدارة المستخدمين
- إدارة الخدمات
- متابعة الطلبات والإيرادات

## التقنيات المستخدمة

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS v4, shadcn/ui
- **Database**: PostgreSQL (Supabase)
- **Authentication**: Supabase Auth
- **Deployment**: Vercel

## البنية التقنية

### قاعدة البيانات

```
profiles - معلومات المستخدمين والأدوار
services - الخدمات المعروضة
orders - الطلبات والمعاملات
wallets - محافظ البائعين
wallet_transactions - سجل المعاملات المالية
reviews - التقييمات والمراجعات
conversations - المحادثات
messages - الرسائل
```

### الصفحات الرئيسية

```
/ - الصفحة الرئيسية
/search - البحث عن الخدمات
/services/[id] - تفاصيل الخدمة
/services/[id]/order - صفحة الطلب
/orders - قائمة الطلبات
/orders/[id] - تفاصيل الطلب
/messages - المحادثات
/messages/[id] - صفحة المحادثة
/dashboard - لوحة التحكم
/seller/services - خدمات البائع
/seller/wallet - محفظة البائع
/seller/reviews - تقييمات البائع
/admin - لوحة الإدارة
/auth/login - تسجيل الدخول
/auth/signup - إنشاء حساب
```

